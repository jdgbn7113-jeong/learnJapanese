문장 분석 ㉘

けれども個人の別荘はそこここにいくつでも建てられていた。

1. 히라가나 변환

けれども こじんの べっそうは そこここに いくつでも たてられていた。

2. 전체 의미

그러나 개인 별장은 여기저기에 얼마든지 세워져 있었다.

3. 한자 단어의 의미와 발음

한자발음의미
個人こじん개인
別荘べっそう별장
建てるたてる세우다, 짓다

4. 문법 분석

요소설명
けれども역접 접속사. "그러나, 하지만"
そこここに「そこ」(거기) + 「ここ」(여기) 반복. "여기저기에"
いくつでも「いくつ」(몇 개) + 「でも」(~라도). "얼마든지, 몇 개라도"
建てられていた「建てる」(짓다) + 「られる」(수동) + 「ていた」(과거 상태). "지어져 있었다"

5. 특이점

「そこここ」: 「あちこち」(여기저기)의 문어체·고풍스러운 표현입니다. 「そこ」와 「ここ」를 직접 붙인 것으로, 소설의 서술적 어조에 어울립니다.
시대적 배경: 메이지·다이쇼 시대에 가마쿠라는 도쿄의 부유층과 지식인들이 여름 별장을 짓는 인기 휴양지였습니다. 화자가 머무는 곳은 외진 시골이지만, 주변에는 부유층의 별장이 많았다는 대비가 드러납니다.
수동태의 기능: 「建てる」(능동: 짓다)의 수동형 「建てられる」로 주체를 명시하지 않고 "건물이 지어진 상태"를 묘사합니다. 일본어에서 자주 쓰이는 "상태 수동" 용법입니다.
